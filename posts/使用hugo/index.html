hugo结合githubpage部署个人网站

2021.03.12

hugo

下载hugo

选择指定版本 https://github.com/gohugoio/hugo/releases

使用Hugo

hugo new site /path/to/quickstart
cd /path/to/quickstart
hugo server #默认localhost:1313，访问即可

选择主题themes

例如：git clone https://github.com/imzeuk/hugo-theme-zozo themes/zozo

git clone https://github.com/kakawait/hugo-tranquilpeak-theme.git themes/tranquilpeak

该主题的文档参考：https://github.com/kakawait/hugo-tranquilpeak-theme/blob/master/docs/user.md#tranquilpeak-configuration

将zozo下的examplesite 所有文件复制到 quickstart下
然后修改标题等，自定义信息

打包生成

hugo # 打包到public目录

hugo中文

参考:https://www.gohugo.org/

githubPage使用

参考：https://pages.github.com/

上传到github

#接hugo
cd public
git init
git remote add origin 你的仓库地址
git add .
git commit -m"description"
git push -u origin master

打开yourname.github.io
