toml语法

0001.01.01

介绍

值可以是以下形式字符串、整数、浮点数、布尔值、时间、数组或内联表

字符串

多行字符串

str1 = """
文本内容
"""

日期

ldt1 = 1979-05-27T07:32:00

数组

#数组必须内为同一数据类型
arr1 = [1, 2, 3]
arr2 = ["red", "blue", "yellow"]
arr3 = [[1, 2], [3, 4, 5]]

表格[字典 map]

[tab-1]
key1 = "abc"
key2 = 321

#多级嵌套
[a.b.c]
key = "hehe"

内联表

name = { first = "Tom", last = "Preston-Werner" }

表格数组

#相当于列表 数组嵌套字典

[[list]]
name = "Hammer"
sku = 738594937
 
[[list]]
 
[[list]]
name = "Nail"
sku = 284758393
color = "gray"

对应的json:

{
  "products": [
    { "name": "Hammer", "sku": 738594937 },
    { },
    { "name": "Nail", "sku": 284758393, "color": "gray" }
  ]
}

注意

TOML 文件必须使用 .toml 来作为其扩展名
